    Prefix

    Prefix is a syllable which comes before any words and changes in meeaning.

  • Pre means before and Fix means attach.

    • Ex.

  • Non + living → Nonliving
  • Un + happy → Unhappy
  • Im + possible → Impossible
  • Pre + position → Preposition
  • Vice + president → Vicepresident

    • Suffix

    Suffix is a syllable which comes after words and changes in meaning.

    Ex.

  • Child + hood → Childhood
  • Beauty + ful → Beautyful
  • Slow + ly → Slowly
  • Attach + ment → Attachment

    • Compund Word

    A word which is made up two or more than two words is called Compound Word .

    Ex.

    Father-In-Law, Land-Lord, Classification

    Modern Defination

    • A Noun is a naming word.
    • Pronoun is a replacing word.
    • Adjective is a qualifying word.
    • Verb is a doing word.
    • Adverb is a modifying word.
    • Preposition is a relating word.
    • Conjunction is a joining word.
    • Interjunction is a exclaming word.
